# Metascan Pro

**High-Fidelity Element Inspector & Metadata Extractor for Obsidian.**

Metascan Pro is an elite developer utility designed to provide deep visibility into the Obsidian DOM. It allows users to precisely select any UI element, capture its underlying metadata, and trace its attributes with zero-scroll data density.

## ✨ Features

- **Precision Selection**: Frame-accurate hover highlighting and click-to-grab selection.
- **Deep Metadata Trace**: Extracts tag names, IDs, classes, and inner text for any DOM node.
- **Elite UI**: Obsidian-native design with OKLCH Vivid Electropurple tokens.
- **Zero-Scroll Density**: Optimized data presentation for high-information density without vertical scroll fatigue.
- **Atomic Deployment**: Built using the Metascan Factory Hub for zero-config distribution.

## 🚀 Installation

### Manual Installation
1. Download the latest `main.js` and `manifest.json` from the [Releases](https://github.com/beto-group/metascan-pro/releases) page.
2. Create a folder named `metascan-pro` in your vault's `.obsidian/plugins/` directory.
3. Move the downloaded files into that folder.
4. Enable the plugin in Obsidian's **Community Plugins** settings.

### Via BRAT
1. Install the [Obsidian BRAT](https://github.com/TfTHacker/obsidian42-brat) plugin.
2. Add the repository URL: `https://github.com/beto-group/metascan-pro`.
3. Metascan Pro will be automatically installed and kept updated.
